A Little About Me And My Website.


I am a Indie Game Developer that does projects across many different engines including - Godot, Roblox Studio and Unreal Engine. It is very rare I release a project but when I do, I make sure that it is in a good state for release. I started my Game Developing journey when I was 12 and decided I wanted to learn how to make games. I started off using simple engines like Scratch and even made a few projects using PyGame. I remember how mind blowing it was to see that I had made a Snake Game from complete scratch in my own bedroom and from there, I started to learn more until eventually I started making projects for the public and that is what this website is for. This website is for me to upload my Games and Music all in one place for my supporters. I make sure to include download links or embed my games ocassionally.
